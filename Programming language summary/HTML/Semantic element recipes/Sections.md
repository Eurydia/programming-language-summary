## Common practices

### Sections as chapters

Sections can be imagined as chapters of a book.
Each chapter contains a heading and some content.

## Recipes

### Simple section

A simple section contains a `h1` element and a paragraph.

```html
<section>
	<h1></h1>
	<p></p>
</section>
```
